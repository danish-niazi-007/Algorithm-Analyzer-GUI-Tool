# Algorithm Analyzer GUI 🚀

An interactive, visualization-based graphical user interface (GUI) tool built with Python to demonstrate the working of complex algorithms in real-time. This project currently features visualizer modules for **Heap Sort** and **Kruskal's Algorithm**, providing a clean, modern, and dark-themed UI using CustomTkinter.

## ✨ Features

- **Real-Time Visualization:** Watch how algorithms process data and make decisions step-by-step.
- **Heap Sort Visualizer:** Generates a random array of bars and visualizes the heapifying and sorting process.
- **Kruskal's Algorithm Visualizer:** Generates a random set of nodes and weighted edges, visualizing the Minimum Spanning Tree (MST) creation using the Union-Find logic.
- **Modern UI/UX:** Built with `CustomTkinter` for a sleek, dark-mode aesthetic.
- **Interactive Controls:** Easily generate new datasets/graphs and trigger algorithms with intuitive buttons.

## 🛠️ Tech Stack

- **Language:** Python 3.x
- **GUI Framework:** CustomTkinter (built on top of Tkinter)
